Laboratory Work

Labs 1 – 5

Lab 1

User Interface (UI)

In Lab 1, I was introduced to building basic Android UI layouts using Jetpack Compose. I learnt how to structure composable functions and arrange elements such as Text, Image, and Column on the screen. This lab gave me a solid foundation in designing clean and readable interfaces for mobile applications.

Lab 2

Interaction & Events

Lab 2 focused on handling user interactions such as button clicks, text input, and state changes using remember and mutableStateOf. I discovered how Compose re-composes the UI automatically when state updates, making the development process intuitive. This lab helped me understand the reactive nature of Jetpack Compose.

Lab 3

Material Design

In Lab 3, I applied Material Design 3 components such as TopAppBar, FloatingActionButton, and Cards to create a polished and consistent UI. I learnt how theming and colour schemes work in Compose to give apps a professional look and feel. Seeing a plain layout transform into a visually appealing screen was very satisfying.

Lab 4

Navigation

Lab 4 introduced multi-screen navigation using NavController and NavHost in Jetpack Compose. I built an app with multiple screens and learnt how to pass data between them safely using navigation arguments. Understanding navigation was a turning point that allowed me to build more complete, real-world applications.

Lab 5

Room Database

Lab 5 covered local data persistence using Room Database, including setting up Entity, DAO, and Database classes. I learnt how to perform CRUD operations and observe data changes in the UI using Flow. This lab was the most challenging but also the most rewarding, as I could finally build apps that remember user data.

Project Showcase

Projects

Project 1

HeroMeal — Discounted Food Finder App

Problem Statement

Food waste is a growing issue in Malaysia, with large amounts of unsold food being discarded daily by restaurants and food stalls. To help solve this problem, I developed HeroMeal — a mobile application that allows users to find discounted leftover food from nearby restaurants, helping reduce food waste and supporting affordable food access aligned with SDG 2: Zero Hunger.

Lessons Learned

Building HeroMeal strengthened my skills in Jetpack Compose navigation and UI design across multiple screens. I learnt how to manage app state effectively and structure a real-world project with clean separation of concerns. This project showed me that even a student-built app can address meaningful social problems.
